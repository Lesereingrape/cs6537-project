# 对外协作接口说明

本文件定义成员 B 系统与其他成员的最小交互接口，仅包含输入/输出约定。

## 1) 与成员A的接口（内容输入）

- 输入目录：`data/products/`
- 输入格式：UTF-8 编码 `.md` 文件
- 建议命名：`<category>_<variant>.md`
- 最小字段建议：商品名、价格区间、关键属性、适用场景

系统行为：
- 运行 `python src/main.py --build-index` 时自动读取并入库。

## 2) 与成员D的接口（实验提示词输入）

- 调用方式：命令行参数 `--query` + `--top-k`
- 示例：`python src/main.py --query "预算300内，推荐收纳箱" --top-k 4`

系统行为：
- 将查询做语义检索并输出导购建议文本。

## 3) 与成员C的接口（评估数据输出）

- 输出载体：标准输出（控制台）
- 输出内容：
  1. 检索片段预览（含来源文件名）
  2. Agent 推荐结果（自然语言）

建议：
- 成员C可将输出复制到其数据记录模板中完成多维评估。
